## Installation

### Clone

Clone the repo to your local machine: `git clone https://github.com/kevinmjacobs/kevinsays`

### Setup

- Install all package dependencies

```shell
npm install
```

- Build the webpack

```shell
npm run build
```

- Start the app

```shell
npm start
```
